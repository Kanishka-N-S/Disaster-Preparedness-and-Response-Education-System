## Disaster-Preparedness-and-Response-Education-System

A web-based Disaster Education and Response System  that helps users learn safety measures, track preparedness, and simulate emergency responses through modules, quizzes, a survival game, and a map-based panic alert system. 

## 🚀 Features

### 👤 User Authentication
- Register/Login system (Student & Teacher roles)
- Data stored using browser localStorage

### 📊 Dashboard
- Track learning progress
- Quick navigation to all modules

### 📚 Learning Modules
- Earthquake Safety
- Flood Preparedness
- Fire & Evacuation
- Step-by-step guidance (Before, During, After)

### 🧠 Quiz System
- Interactive MCQ-based quizzes
- Score tracking
- Badge rewards

### 🎮 Survival Game
- Mini simulation game
- Reach safe zone while avoiding obstacles
- Enhances decision-making skills

### 📖 Knowledge Base
- Emergency guides & safety tips
- Search functionality
- Downloadable safety poster

### 🗺️ Map & Panic System
- Live location tracking
- Safe shelter markers
- Panic alert simulation 🚨

### 👨‍🏫 Teacher Dashboard
- Monitor student progress
- Send alerts to students
- Award badges

### 🏅 Profile & Progress
- Readiness percentage
- Earned badges
- Performance tracking
